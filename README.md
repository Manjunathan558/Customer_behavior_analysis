# Customer_behavior_analysis
This project analyzes customer shopping behavior data to uncover purchasing patterns, customer preferences, and business insights. The analysis was performed using Python, SQL, and Power BI to transform raw customer data into meaningful visualizations and actionable recommendations. 

## Project Overview

This project focuses on analyzing customer purchasing behavior to uncover patterns, trends, and business insights. Using Python, SQL, and Power BI, the dataset is cleaned, explored, and visualized to help understand customer preferences, spending habits, and purchasing decisions.

## Objectives

* Analyze customer demographics and purchasing behavior.
* Identify top-selling products and categories.
* Evaluate customer spending patterns.
* Discover factors influencing purchase decisions.
* Create interactive dashboards for business insights.

## Dataset Information

The dataset contains customer-related information such as:

* Customer ID
* Age
* Gender
* Location
* Item Purchased
* Category
* Purchase Amount
* Review Rating
* Subscription Status
* Payment Method
* Frequency of Purchases

## Project Workflow

### 1. Data Collection

* Imported customer behavior dataset in CSV format.

### 2. Data Cleaning

* Checked for missing values.
* Removed duplicates.
* Corrected data types.
* Handled inconsistent records.

### 3. Exploratory Data Analysis (EDA)

* Customer demographics analysis.
* Revenue analysis by gender.
* Purchase trends across categories.
* Product rating analysis.
* Spending behavior analysis.

### 4. SQL Analysis

Performed SQL queries to answer business questions:

* Total revenue by gender
* Top-selling products
* Average product ratings
* Purchase frequency analysis
* Customer segmentation

### 5. Dashboard Development

Created a Power BI dashboard containing:

* Total Revenue KPI
* Customer Distribution
* Revenue by Category
* Purchase Frequency Analysis
* Product Rating Insights
* Gender-wise Spending Analysis

### 6. Reporting

Generated insights and recommendations based on customer behavior patterns.

## Key Insights

* Identified the highest revenue-generating customer segments.
* Determined the most frequently purchased products.
* Analyzed customer satisfaction through review ratings.
* Evaluated purchasing trends across different demographics.
* Discovered opportunities for targeted marketing strategies.
